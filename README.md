treclip
=======

> Clipboard assistent for Trello checklists from bookmarks

Dependencies
------------

- bash
- sqlite3
- xclip
- bookmarker - [_my fork_](/alexandre-mbm/bookmarker)
- yad
- wmctrl

Use
---

```console
$ export PATH=$PATH:DIRECTORY/bookmarker
$ sh DIRECTORY/treclip/treclip.sh
```
